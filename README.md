其中專題之核心概述

### 專題名稱：反應時間測試遊戲系統

核心宗旨：透過「遊戲化」的方式，解決傳統按鈕測試枯燥、缺乏視覺回饋與數據分析的缺點，打造一個具備量化測量與即時視覺回饋的反應力測試系統。

### 遊戲流程與系統運作

系統完美結合了軟硬體，完整的互動流程如下：

身分觸發：使用者透過 RFID 磁扣感應，啟動遊戲。

隨機倒數：LED 燈進行 2~4 秒的隨機倒數，防止使用者預判。

反應測量：倒數結束 LED 亮起，使用者迅速伸手，超音波距離感測器偵測到物體接近至特定距離即算完成，紀錄毫秒級反應時間。

影像捕捉：系統同步觸發手機鏡頭，利用 OpenCV 拍攝參賽者大頭照。

數據上傳與展示：ESP32 將成績與照片透過 HTTP POST 傳至後端，並由 Node-RED 渲染出包含大頭照的即時英雄榜（排行榜）。

### 技術架構與使用工具

本系統採用了前後端分離與物聯網 (IoT) 架構：

硬體層 (Sensor & Control)：

ESP32 微控制器 (作為核心運算與網路傳輸橋樑)

超音波距離感測器、RFID 偵測器、LED 燈

手機攝像頭

軟體與後端層 (Data & API)：

Django REST API：處理硬體傳來的數據請求

MySQL：資料庫系統，儲存反應時間與使用者資料

OpenCV：處理與記錄捕捉到的影像

Node-RED：進行視覺化的排行榜網頁展示

### 系統優勢與應用場景

主要突破：相比傳統電腦程式，本系統具備高精度測量、自動化綁定臉部影像（方便辨認成績歸屬）、完整數據庫紀錄，且擴展性極強。

未來應用潛力：

競技與訓練：運動員敏捷度訓練、電競玩家反應力評估。

安全與醫療：職業駕駛員疲勞/反應測試、醫療復健患者的進度追蹤。
